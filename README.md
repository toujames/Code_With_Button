# Code_With_Button (Senior 2 Group 2B Project)

This code outputs the speed of an object using the doppler effect

## Main Components
* Bushnell Receiver
* LCD Display with Backpack
* Arduino

## Connection

* IF Signal to A0 in arduino

### SPI Backpack
* DAT pin is #6
* CLK pin is #2
* LAT pin is #4
